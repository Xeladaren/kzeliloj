
# Kzeliloj

## Brief

Librairie for basics tools.

## Tools List

- **list :** Doubly linked list.

## Compil

Make lib :
```bash
make all
```

Make example progs :
```bash
make example
```

## Install

Install file in `/usr/lib/`:
```bash
sudo make install
```

Install file in custom dir:
```bash
INSTALL_DIR=/custom/dir make install
```
